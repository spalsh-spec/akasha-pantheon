The fuzzy-truth oracle.

A local-first AI interface that holds three legitimate truths at once — without science-washing the myth or myth-washing the science.

The problem

One-answer AI is dishonest about most real questions.

When the question is empirical (mass of an electron) — one answer is correct. When the question lives between empirical, mythic, and synthetic registers — most questions a human actually asks — collapsing it to one answer either picks an authority and pretends, or hedges into uselessness.

"Why does π appear in probability?" "What did the builders of Göbekli Tepe know?" "What is the physics of flow state?" None of these collapse cleanly.

03 / 12

The insight

Hold three truths at once.

  • Literal — what consensus science says, with citations.
  • Mythic — how a named tradition (Vedic / Hermetic / Yoruba / Hopi) framed it.
  • Fuzzy — the synthesis when both are taken seriously.
  • Plus a Sigil (3–7 word symbol) and a Tweet (240-char compression).
  • Plus a self-reported Confidence 0.00–1.00 on every answer.

How it works

Format-as-prior.

The 5-section format is enforced by the system prompt and verified at parse time. Outputs that don't match the format are not rendered as answers. This is a structural intervention in the LLM's output distribution — it forces three parallel registers to be filled, simultaneously.

LITERAL: Pi (π ≈ 3.14159) arises in probability through…
MYTHIC:  In Kabbalah, this is Ein Sof, the boundless circle…
FUZZY:   Geometry as a probabilistic ghost…
SIGIL:   Circles within circles, infinite recursion.
TWEET:   Pi's everywhere — geometry's a probabilistic ghost.
CONFIDENCE: 0.85

Two modes, one toggle

FAST (local, private) ↔ DEEP (cloud, free).

  • FAST · Ollama gemma3:4b · runs entirely on the user's M1 16GB · zero network egress · zero cost · ~13s/answer.
  • DEEP · Cloudflare Workers AI Llama 3.3 70B FP8 Fast · free tier · ~5s/answer · larger model, better calibration.
  • User toggles per-question. Default DEEP, fall-through to FAST if cloud is unavailable.

What's novel

Five claims, all defensible.

  • Epistemic pluralism, operationalised. The 5-section format is a Bayesian-style structural prior on the output distribution.
  • Compulsory tradition naming. The MYTHIC field rejects vague "ancient wisdom" — the model must name a specific tradition.
  • Self-reported confidence as first-class output. Auditable across questions and across models.
  • Local-first sovereignty. Default mode requires no key, no cloud, no telemetry.
  • iCloud-archived as markdown. Every answer is a portable, searchable file.
⟁AKASHA · PANTHEON
09 / 12

What it isn't

Stated upfront because honesty is the moat.

  • Not a forecaster. Asked to predict the next 5 stages of a war on specific dates, both models refused to fabricate and dropped confidence to 20%. That is the right answer. No model has live intel.
  • Not a chat assistant. Format-locked output is a poor fit for narrative or code generation. Use general LLMs for those.
  • Not making metaphysical claims. The MYTHIC field is a framing, not an assertion that Kabbalah is empirically true.

A real answer (Llama 3.3 70B · 7.9s)

"What is the physics of flow state?"

Flow state is characterised by heightened focus and productivity, linked to optimal challenge-skill ratios as proposed by Csíkszentmihályi (1975), and involves neurochemicals like dopamine. LITERAL · cited tradition: Csíkszentmihályi 1975
In Taoist tradition, flow state is akin to wu-wei — effortless action — where the individual is fully immersed, transcending conscious thought, as described in the Tao Te Ching. MYTHIC · named tradition: Taoist
Optimal challenge sparks dopamine, igniting flow state. TWEET · 240-char compression · confidence 0.85
